# django_messageboard
## 项目工具
* Django2.0 python3.6 bootstrap4
## 项目简介
以Django框架为基础的留言板demo，具体内容包括用户发布文章，添加评论，管理文章包括编辑，新增，删除，还包括用户密码的重置及更改，并使用邮件服务器发送邮件。大量使用了内置的视图。前端使用bootstrap进行美化
**感谢Will Vincent的博客，非常感谢**
